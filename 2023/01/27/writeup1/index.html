Challenge Description

We are given a netcat session that when we connect to we get a shell as the user user

Our goal is to escalate our privileges and get the flag located in /root/flag.txt .

There is a README.md file inside our current directory, when we try to cat it we get an error

seems like it is replacing the space ' ' with nothing '' so every command we try to execute that contains spaces won’t work. So how can we bypass the space in this case ?

If we do a little research we can find that in order to bypass the space in the Linux command line we can use the ${IFS} like they mentioned here

https://unix.stackexchange.com/questions/351331/how-to-send-a-command-with-arguments-without-spaces

Now that we bypassed the space filter let’s cat the README.md file again using cat${IFS}README.md
